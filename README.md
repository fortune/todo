# TODO アプリ

シンプルな TODO アプリです。依存ライブラリなし、ビルド不要で動作します。

## 使い方

`index.html` をブラウザで開くだけで使えます。

## 機能

- タスクの追加（Enter キーまたは「追加」ボタン）
- タスクの完了/未完了の切り替え
- タスクの削除
- フィルター（すべて / 未完了 / 完了済み）
- 完了済みタスクの一括削除
- データは `localStorage` に保存（リロード後も保持）

## 構成

```
├── index.html   # HTML 構造
├── style.css    # スタイル
└── app.js       # アプリロジック
```
